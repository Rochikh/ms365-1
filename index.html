Services Microsoft 365 Expliqués

Comprendre Teams, SharePoint & OneDrive

Que sont ces services Microsoft 365 ?

Microsoft 365 offre une suite d'outils de productivité qui fonctionnent ensemble. Voici comment Teams, SharePoint et OneDrive diffèrent et se complètent.

Microsoft Teams

Le hub pour le travail d'équipe et la communication dans Microsoft 365

  • Communication par chat
  • Réunions et appels vidéo
  • Conversations par canaux
  • Intégrations d'applications
  • Partage et collaboration sur les fichiers

SharePoint

Plateforme de gestion de contenu et de collaboration d'équipe

  • Sites d'équipe et intranet
  • Bibliothèques de documents
  • Listes et gestion des données
  • Automatisation des flux de travail
  • Publication de contenu

OneDrive

Stockage cloud personnel pour les fichiers de travail

  • Stockage de fichiers personnels
  • Synchronisation entre appareils
  • Accès hors ligne aux fichiers
  • Partage avec des personnes spécifiques
  • Sauvegarde des dossiers importants

Comment ils fonctionnent ensemble

Microsoft Teams

Le hub de collaboration

SharePoint

Stockage du contenu d'équipe

OneDrive

Stockage du contenu personnel

Lorsque vous créez une équipe dans Microsoft Teams, un site SharePoint est automatiquement créé pour stocker les fichiers de l'équipe. Vos fichiers personnels sont stockés dans OneDrive, mais vous pouvez les partager et collaborer dessus via Teams également.

Travailler dans Teams

Lorsque vous partagez des fichiers dans un canal Teams, ils sont stockés dans le site SharePoint de l'équipe. Les fichiers des chats privés sont stockés dans votre OneDrive.

Accès de n'importe où

Vous pouvez accéder à vos fichiers directement via SharePoint ou OneDrive, ou de manière transparente via Teams.
